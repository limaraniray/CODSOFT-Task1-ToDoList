## 📖 About the Project

A **To-Do List application** is a useful tool that helps users manage and organize their tasks efficiently. This project is a command-line-based application using Python, allowing users to create, update, and track their to-do lists in a simple and clean interface.

This is **Task 1** of the CodSoft Python Programming Internship.

---

## ✨ Features

| Feature          | Description                              |
| ---------------- | ---------------------------------------- |
| ➕ Add Task      | Add a new task with a title              |
| 📋 View Tasks    | View all tasks with their current status |
| ✔️ Mark Complete | Mark any pending task as done            |
| ✏️ Update Task   | Edit the title of an existing task       |
| 🗑️ Delete Task   | Remove a task from the list              |
| ❌ Validation    | Handles invalid inputs gracefully        |
| 🔄 Loop Menu     | Continuous menu until user exits         |

## ▶️ How to Run

### Prerequisites

- Python 3.x must be installed → [Download Python](https://www.python.org/downloads/)

### Steps

```bash
# Step 1: Clone the repository
git clone https://github.com/limaraniray/CODSOFT-Task1-ToDoList.git

# Step 2: Navigate into the folder
cd CODSOFT-Task1-ToDoList

# Step 3: Run the application
python task1_todo_list.py
```

## 💻 Usage

```
========================================
        📝 TO-DO LIST APP
========================================
1. Add Task
2. View All Tasks
3. Mark Task as Complete
4. Update Task
5. Delete Task
6. Exit
========================================
Choose an option (1-6):
```

### ➕ Adding a Task

```
Choose an option (1-6): 1
Enter task title: Complete Python Internship
✅ Task 'Complete Python Internship' added successfully!
```

### 📋 Viewing Tasks

```
Choose an option (1-6): 2
--- Your Tasks ---
1. [⏳ Pending] Complete Python Internship
2. [✔ Done   ] Push code to GitHub
```

### ✔️ Marking Complete

```
Choose an option (1-6): 3
Enter task number to mark complete: 1
✅ Task 'Complete Python Internship' marked as complete!
```

### 🗑️ Deleting a Task

```
Choose an option (1-6): 5
Enter task number to delete: 1
🗑️ Task 'Complete Python Internship' deleted!
```
